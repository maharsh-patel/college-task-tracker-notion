# College Task Tracker (Notion)

A minimal task tracking system designed for college students to manage assignments, labs, exams, and daily priorities with clarity.

## Why I built this
Managing multiple deadlines across subjects was getting messy. I needed a simple system that highlights what actually needs attention each day instead of adding more complexity.

## Features
- Single master task database
- Automatic days-left calculation
- Deadline-based urgency levels
- Clear daily and weekly focus
- Minimal and distraction-free design

## How to use
1. Open the Notion template link below
2. Click **Duplicate**
3. Add it to your Notion workspace
4. Start adding your college tasks
